Introduction About HTML


It is a standard markup language for creating webpages. HTML basically describes the meaning and structure of webpage. It describes the webpage appearance, presentation. HTML consists of series of different elements and these elements tells the browser how to display the content.


THIS IS HOW A SIMPLE HTML DOCUMENT LOOKS LIKE!

<!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
</head>
<body>
<h1>Heading</h1>
<p>This is my doucment's paragraph</p>
</body>
</html>

OutPut




Explation of Above Tags

The above examples explain the use of different elements and how these tags and elements gives the structure to our webpage.

<!DOCTYPE html>
This declaration defines that this document is an HTML5 document.

<html>
It is the root element of an HTML page.

<head>
It contains the meta information of the HTML page.

<title>
This element specifies a title for the HTML page. The title is shown in the browser title bar or in the tabâ€™s page.

<body>
This element defines the document's body. It is a container for all the visible contents, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.

<h1>
This element defines a large heading.

<p>
This element defines a paragraph.

We are reading this term element or html elements since the start so what is basically these HTML elements?

An HTML element is defined by a start tag, some content, and an end tag: The content The HTML element is everything from the start tag to the end tag! Syntax or structure of an HTML page. Root element should always be used in the starting of code. Contains meta data. Contains all the content of your webpage. NOTE: Without these tags you cannot form your code structure. The syntax has to be followed in same sequence so as to form a proper structure of your webpage. Further we will learn how to add and display content on web browser using these tags !
